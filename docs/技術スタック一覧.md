# 朝イチ気分チェック — 技術スタック一覧

このプロジェクトで使っている技術をまとめました。

---

## 一覧（役割別）

| 役割 | 技術 | 用途・補足 |
|------|------|------------|
| **フロントエンド** | HTML | 画面の骨組み（見出し・ボタン・グラフの枠など） |
| | CSS | 見た目（色・余白・レイアウト） |
| | JavaScript（素のJS） | 動き・API 呼び出し・グラフ描画。フレームワークなし。 |
| **バックエンド** | Python 3 | プログラミング言語 |
| | Flask | Web サーバー・API（/api/today, /api/mood, /api/moods） |
| | gunicorn | 本番環境で Flask を動かすサーバー（Render 用） |
| **データベース** | SQLite | 気分の記録（日付・点数）を保存。ファイル 1 つ（mood.db） |
| **バージョン管理** | Git | コードの履歴管理 |
| **コードの保管** | GitHub | リポジトリのホスティング。Render がここから取得 |
| **デプロイ・インフラ** | Render | クラウド上でアプリを動かし、公開 URL を提供 |
| **ローカル環境** | macOS | 開発・実行環境 |
| | launchd | ログイン時にサーバー起動＋ブラウザを開く（オプション） |
| | Python venv | プロジェクト専用の Python 環境（.venv） |

---

## 図：レイヤーごと

```
  【ユーザーが触る】
       │
  ┌────┴────┐
  │ ブラウザ  │  ← HTML / CSS / JavaScript
  └────┬────┘
       │ HTTP（API 呼び出し）
  ┌────┴────┐
  │ サーバー  │  ← Python / Flask（本番では gunicorn）
  └────┬────┘
       │ 読み書き
  ┌────┴────┐
  │   DB    │  ← SQLite（mood.db）
  └─────────┘

  【コードの保管】  GitHub（Git）
  【本番の実行場所】 Render
  【ローカル】       macOS, venv, launchd
```

---

## ファイル・技術の対応

| 技術 | 主なファイル・場所 |
|------|---------------------|
| HTML | static/index.html |
| CSS | static/style.css |
| JavaScript | static/app.js |
| Python / Flask | app.py |
| SQLite | mood.db（自動生成） |
| 依存ライブラリ一覧 | requirements.txt（Flask, gunicorn） |
| 本番起動 | Render の Start Command: gunicorn --bind 0.0.0.0:$PORT app:app |

---

## 一言メモ

- **フロント** … ビルドツールなし。そのままブラウザで読む HTML/CSS/JS。
- **バックエンド** … Flask で API を提供。ローカルでは `python app.py`、本番では gunicorn。
- **DB** … サーバー不要の SQLite。1 ファイルで完結。
- **インフラ** … ローカルは「自分の Mac」、本番は「Render のサーバー」。
